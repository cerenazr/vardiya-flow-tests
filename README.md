# Vardiya Flow – Cypress Test Suite

Bu repo, Lovable tabanlı *Vardiya Flow* projesi için yazılmış E2E test senaryolarını içerir.

## Çalıştırma
npm install
npx cypress run   # headless mod
npx cypress open  # GUI mod

## Test Kapsamı
- Login ve kullanıcı akışları
- Shift (vardiya) yönetim senaryoları
- API yanıt doğrulamaları

## Raporlama
Cypress HTML raporları, video ve ekran görüntüsü çıktıları otomatik olarak  ve  klasörlerine kaydedilir.
